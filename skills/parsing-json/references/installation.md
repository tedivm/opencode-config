## Installing jq

### macOS

```bash
brew install jq
```

### Linux (Debian/Ubuntu)

```bash
sudo apt install jq
```

### Linux (RHEL/CentOS/Fedora)

```bash
sudo dnf install jq
# or
sudo yum install jq
```

### Linux (Arch)

```bash
sudo pacman -S jq
```

### Linux (Alpine)

```bash
apk add jq
```

### Windows

- **WSL**: Use the Linux instructions above
- **Chocolatey**: `choco install jq`
- **Scoop**: `scoop install jq`
- **Direct download**: <https://github.com/jqlang/jq/releases>

### Verify Installation

```bash
jq --version
```
