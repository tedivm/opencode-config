## Workflow

1. Check if `openspec` is installed by running `which openspec`. If not found, install it using one of:
   - `brew install openspec`
   - `npm install -g @fission-ai/openspec`
2. Run `openspec init --tools github-copilot --force` to initialize non-interactively
3. Move only the `openspec-*` skills from `.github/skills/` to `.agents/skills/`:
   - Run `mkdir -p .agents/skills`
   - For each directory in `.github/skills/` matching `openspec-*`, move it to `.agents/skills/`
   - Remove the `.github/skills/` directory if it is empty
4. Delete OpenSpec prompts from `.github/prompts/`:
   - Remove only the `opsx-*.prompt.md` files generated by OpenSpec
   - Leave any other files in `.github/prompts/` untouched
   - Remove the directory if empty
